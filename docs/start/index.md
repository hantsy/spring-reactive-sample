# Getting Started

This section teaches how to build Spring WebFlux applications both from scratch and with Spring Boot 4. Follow the step-by-step tutorials below:

- Introduction — core concepts and runtime choices
- Create a WebFlux app from scratch — manual wiring to learn the fundamentals
- Create a WebFlux app with Spring Boot — the fastest path to a running app

Each tutorial is standalone and includes runnable examples and configuration notes.
